# Как запустить проект в VS Code

Этот проект создан с использованием React, TypeScript, Vite и Tailwind CSS.

## Инструкция по запуску

1.  **Скачайте файлы**: Скопируйте все файлы из этого проекта в папку на вашем компьютере.
2.  **Установите Node.js**: Убедитесь, что у вас установлен Node.js (версия 18 или выше).
3.  **Откройте терминал**: Перейдите в папку проекта.
4.  **Установите зависимости**:
    ```bash
    npm install
    ```
5.  **Запустите проект**:
    ```bash
    npm run dev
    ```
6.  **Откройте в браузере**: Перейдите по ссылке, которая появится в терминале (обычно http://localhost:3000).

## Структура проекта

*   `/src`: Исходный код приложения
    *   `/components`: Переиспользуемые компоненты (карточки, модальные окна)
    *   `/screens`: Основные экраны (День, Неделя, Профиль)
    *   `/types.ts`: TypeScript типы
    *   `/utils.ts`: Вспомогательные функции
    *   `/mockData.ts`: Тестовые данные (расписание, профиль)
*   `App.tsx`: Главный компонент приложения
*   `tailwind.config.js`: Конфигурация стилей (встроена в index.html или vite config)

## Рекомендации

*   Используйте расширение **Tailwind CSS IntelliSense** для VS Code.
*   Используйте **ESLint** и **Prettier** для форматирования кода.
