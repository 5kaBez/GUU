# Telegram Bot для просмотра расписания

## Описание

Telegram бот позволяет студентам быстро и удобно получить расписание своих занятий через простой процесс выбора фильтров.

## Как использовать бот

### 1. Запуск бота

```bash
python run_bot.py
```

### 2. Использование в Telegram

1. **Начните беседу** с ботом командой `/start`
2. **Выберите форму обучения:**
   - Очная (курсы 1-4)
   - Очно-заочная (курсы 1-5)

3. **Выберите уровень образования:**
   - Бакалавриат (доступен для обеих форм)
   - Магистратура (только для очной формы)

4. **Выберите курс обучения** (в зависимости от формы)

5. **Выберите институт** из 6 доступных:
   - Институт государственного управления и права
   - Институт информационных систем
   - Институт маркетинга
   - Институт отраслевого менеджмента
   - Институт управления персоналом, социальных и бизнес-коммуникаций
   - Институт экономики и финансов

6. **Выберите направление** (загружается из БД на основе выбранных фильтров)

7. **Выберите программу** (загружается из БД)

8. **Выберите номер группы** (загружается из БД)

9. **Получите расписание на сегодня:**
   - Текущая дата и день недели
   - Определение четности недели
   - Список всех пар с указанием:
     - Номера пары и времени
     - Названия предмета
     - Вида пары (лекция, практика и т.д.)
     - Преподавателя
     - Номера аудитории

## Логика работы

### Иерархия фильтров

Все фильтры связаны между собой:

```
Форма обучения
    ↓ (определяет доступные уровни образования)
Уровень образования
    ↓ (определяет доступные курсы)
Курс
    ↓ (вместе с формой, уровнем и институтом определяет направления)
Институт
    ↓
Направление
    ↓
Программа
    ↓
Номер группы
    ↓
Расписание на текущий день
```

### Определение четности недели

- **Неделя 1 начинается:** 9 февраля 2026 года (воскресенье)
- **Неделя 2 начинается:** 16 февраля 2026 года
- И так далее (каждая неделя = 7 дней)

Четность определяется автоматически для текущего дня.

### Расписание

Бот показывает расписание на **текущий день** с учетом:
- Дня недели
- Четности недели
- Выбранной группы

## Переменные окружения

Требуется установить в `.env`:

```
TELEGRAM_BOT_TOKEN=<ваш_токен_бота>
DATABASE_PATH=./data/schedule.db
```

## Логирование

Все события логируются в `logs/bot.log`:
- Запуск бота
- Выбор пользователей
- Ошибки и исключения

## Требования

- Python 3.8+
- pyTelegramBotAPI
- База данных SQLite (data/schedule.db)
